README - Finance Analyzer
FINANCE ANALYZER

Finance Analyzer is a Python-based expense tracking project created to simplify everyday money
management. It helps users record expenses, calculate totals, and generate basic summaries without
needing advanced tools or applications.

FEATURES
1. Add expenses with date, amount, category, and notes.
2. Automatically calculates total expenditure.
3. Provides category?wise summary.
4. Stores all data in a CSV file.
5. Helps visualize total expenditure.

FILES INCLUDED
• finance_analyzer.py  
• expenses.csv  
• README.txt  
• finance_analyzer_PROJECT REPORT.docx

HOW TO RUN
1. Install Python (any version 3.8+ recommended).
2. Download the project folder.
3. Open terminal/cmd inside the folder.
4. Run the script using:  python main.py
